# git_practice
## BASIC
    commit: 파일 추가/ 변경 내용을 저장소에 저장
    push: 파일 추가/ 변경 내용을 저장소에 업로드

## 로컬 저장소/ 원격 저장소
	로컬: 컴퓨터에 직접 저장
	원격: 네트워크 서버에 저장
## BRANCH
	코드의 기능 추가 혹은 수정 시 여러 버전을 나누어 기록 가능함.
